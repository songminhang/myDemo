# 旅游信息后台管理系统
        
        该项目为本人学习SSM框架时练手的项目

# 技术选型

        后端：
        1.Spring ：业务层
        2.Mybatis ：持久层
        3.SpringMVC ：表现层
        4.Spring security ：资源权限控制/安全管理
        5.pagehelper ：分页插件
        6.MySQL ：数据库
        
        前端：
        1.AdminLTE ：前端模板
        2.JQuery

# 内容介绍

        使用Spring security做为延伸扩展的安全功能，结合基本的SSM基本的理论知识和框架模板；
    针对几个表进行简单的CRUD操作练习，以及简单的业务逻辑整合，仅适合学习讨论，也是记录
    自己的第一个单体项目诞生。
         
        至此，准备向分布式微服务方向努力学习，希望有志同道合的人可以提出宝贵意见和讨论
            
# 运行环境

    Tomcat
    Maven
